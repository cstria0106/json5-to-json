# json5-to-json

![Svelte](https://img.shields.io/badge/svelte-%23f1413d.svg?style=flat&logo=svelte&logoColor=white)

Simply convert between various document formats.

[Github Pages](https://cstria0106.github.io/json5-to-json/)

### Run/Build

```sh
pnpm install
pnpm start
pnpm build
```

### Supported formats

- json
- json5
- toml
- yaml
